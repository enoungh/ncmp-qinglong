使用前准备：获取 Cookie

登录网易云音乐

打开 网易云音乐网页版 并登录账号

打开开发者工具

按 F12 打开浏览器开发者工具

获取 Cookie

切换到 Network 选项卡，刷新页面，在请求的 Cookie 中找到 MUSIC_U 和 __csrf 的值

青龙面板部署指南

安装依赖

进入青龙面板 → 依赖管理 → 新建依赖 → 选择 Python3

requests pycryptodome pyncm

配置环境变量

进入青龙面板 → 环境变量 → 新建变量：

变量名 必填 说明
MUSIC_U 必填 MUSIC_U 值
CSRF 必填 __csrf 值
NETEASE_PHONE 必填 登录手机号
NETEASE_MD5_PASSWORD 必填 密码的 MD5 值（或用 NETEASE_PASSWORD）
SCORE 选填 评分策略：1-4（默认 3）
FULL_EXTRA_TASKS 选填 true=完成所有额外任务

拉取脚本仓库

进入青龙面板 → 定时任务 → 新建任务：

ql repo https://github.com/enoungh/ncmp-qinglong.git "main.py|refresh_cookie.py" "tests" "src" "main"

定时规则：0 0 * * *

设置打卡与刷新任务

  • 每日打卡：30 8 * * *
  • 刷新 Cookie：0 22 * * 0

常见问题

遇到验证码/滑块风控？
在电脑浏览器无痕模式下手动登录网易云、完成滑块验证，然后抓取新的 MUSIC_U 和 __csrf 重新覆盖青龙的环境变量即可。
